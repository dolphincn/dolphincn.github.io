git 服务搭建和hexo发布

发表于 2018-06-25 | 更新于 2018-07-26 | 分类于 git

安安装

yum install -y git

添加git服务用户

添加 git repository

mkdir /data/git -p    

git init --bare /data/git/hexo.git

chown -R s_git:s_git /data/git
